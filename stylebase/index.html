Typography

Text

Text inherits configuration from body. Default font is configured as Verdana, "Geneva CE", lucida, sans-serif. All heights are relative to the default font-size which should equal to 1em = 16px and line height, which is also relative, should equal to 1.5em = 20px.

In addition top and bottom margin of all paragraphs and block elements such as h1, h2, h3, h4, h5, h6, blockquote, pre, address and table is configured to 0.5em

Tooltips

A customized tooltip can be displayed by adding class tooltip to any element with data-tooltip attribute present. To display tooltip from the bottom add tooltip-bottom class to the element.

You can also turn tooltip off for separate element with class tooltip-off.

Custom tooltips are purely css3 based feature. If required, tooltips can be turned off by adding following piece of code to your stylesheet.

.tooltip[data-tooltip]:hover:before,
.tooltip[data-tooltip]:hover:after {
	display: none;
}

Buttons

Default buttons

Button styles can be applied with btn class to almost aby element. But you would want to apply them to <a> or <button> elements for best view.

Button Class Description
Button .btn The default button design.
Primary .btn-primary Primary button desing. Should denote the biggest importance of the button action.
Success .btn-success Success button should mean positive button action in response to the user.
Danger .btn-danger Means danger action in the response to the user. User should take the biggest care when performing given action.
Link .btn-link Use this design if you need to display button as a link. Keep in mind you should style this type in the same manner as you style your links.

Button sizes

Buttons can be also styled in three additional sizes in addition to the default one.

Button Class Description
Mini button .btn-mini The smallest button size.
Small button .btn-small Small button size.
Default button N/A Default button size.
Large button .btn-large Largest button size.
